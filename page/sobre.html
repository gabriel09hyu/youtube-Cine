Um YouTube de Filmes e Séries

Olá! Se você chegou até aqui, provavelmente está curioso sobre como este site funciona. Mas com um foco diferente: em vez de vídeos de criadores, o conteúdo é inteiramente sobre filmes e séries.

🎬 Conteúdo Dinâmico com TMDB

Todo o catálogo de "vídeos" que você vê por aqui vem da API do The Movie Database (TMDB). Isso significa que temos um universo de filmes e séries para explorar, com informações, pôsteres e recomendações sempre atualizadas.

🤖 Comentários com IA (Gemini)

A seção de comentários é especial. Para simular uma comunidade ativa, usando a API do Google Gemini para gerar opiniões diversas sobre os filmes. você também pode deixar seu próprio comentário e a IA te responderá, como se fosse outro fã de cinema!

🔥 Feed de "Shorts"

Inspirado nas plataformas de vídeos curtos, criei um feed de "Shorts" que mostra pôsteres de filmes e séries em um formato vertical e viciante. É uma forma rápida e divertida de descobrir novos títulos para assistir.

📂 Gerenciamento Pessoal

Como em um YouTube de verdade, você pode organizar sua vida cinéfila. O site guarda seu histórico de visualizações, permite que você marque os filmes que "Gostou", salve para "Assistir mais tarde" e crie suas próprias playlists.

🔍 Busca e Filtros

Está procurando por algo específico? A barra de busca é totalmente funcional. Além disso, na página inicial, você pode usar os filtros para navegar por categorias como "Populares", "Em Alta" e "Séries (Top)".

📱 Interface Moderna e Responsiva

A interface foi construída do zero para ser limpa, moderna e intuitiva, usando as melhores práticas de CSS. Ela se adapta a diferentes tamanhos de tela, funcionando bem tanto no computador quanto no celular.

Tecnologias Utilizadas

  • HTML5: A base de tudo, estruturando cada página de forma semântica.
  • CSS3: Responsável por todo o visual, com Flexbox e Grid Layout para criar um layout flexível e moderno.
  • JavaScript (ES6+): O cérebro da aplicação, cuidando de toda a interatividade, lógica e comunicação com as APIs.
  • API do The Movie Database (TMDB): A fonte de todo o conteúdo de filmes e séries.
  • API do Google Gemini: A inteligência por trás dos comentários e das respostas da IA.
  • Axios: Para fazer as chamadas às APIs de um jeito mais prático e moderno.
  • LocalStorage: Para salvar suas informações (histórico, playlists, etc.) diretamente no seu navegador.
